Installation

Install directly from the Chrome Web Store with one click — or follow the manual steps below if you prefer.

Recommended — One click install:
Add to Chrome from Web Store

01
Download ZIP

For manual install: download the extension ZIP file and extract it to a folder on your computer.

Download v5.0 ZIP
02
Open Extensions

In Chrome browser, go to the address bar and type:

chrome://extensions

Press Enter to open Extensions page.

03
Enable Dev Mode

In the top-right corner of the Extensions page, toggle on "Developer mode".

Developer mode → ON
04
Load Unpacked

Click "Load unpacked" button that appears after enabling Dev mode. Select the extracted folder.

Done! Extension installed

Extension install karne ke baad kya karna hoga?
Sabse pehle website par login karein. Login karne ke baad extension automatically aapka account detect karega. Credits dashboard par dikhenge — extension mein Customer ID daalkar form fill karo.
Extension safe hai? Data secure rahega?
Haan, bilkul safe. Extension sirf Google Firebase se data read karta hai — aapka hi data, aapki permission se. Koi data third-party ko nahi jaata.
"Load unpacked" ka option nahi dikh raha?
Pehle Developer mode ON karein (extensions page par top-right toggle). Developer mode ON hone ke baad "Load unpacked" button appear hoga.
Extension use karne ke baad credit kitna katega?
Pehli baar ek form fill karne par 1 credit katega. Usi form ka same mapping cache mein save ho jaata hai — dobara fill karne par 0 credits katenge (instant & free)!
Extension Chrome Web Store par available hai?
Haan! Extension ab officially Chrome Web Store par live hai. Upar diye "Add to Chrome — Free" button se seedha install karein — koi developer mode ya ZIP download ki zaroorat nahi.
