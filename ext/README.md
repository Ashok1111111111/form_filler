# AI Form Filler Extension v1.0

Fill government forms automatically with AI!

## Installation
1. Extract this folder
2. Open chrome://extensions/
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select this folder
6. Done!

## Usage
1. Open any form
2. Click extension icon
3. Enter Customer ID
4. Load data
5. Fill form!
